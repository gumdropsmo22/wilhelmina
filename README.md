# Wilhelmina Bot

A Discord bot named Wilhelmina for onboarding, tarot readings, voice-prompt interactions, and admin tools.

---

## Prerequisites

- Node.js (v16 or later)  
- Git  
- MongoDB (for persistence)  
- A Discord application with bot token, client ID, and guild ID
